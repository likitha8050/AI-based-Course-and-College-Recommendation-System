# 🎓 AI-Based College Admission Predictor

An intelligent web application that helps students predict their chances of getting admission into various colleges based on their academic performance, entrance exam scores, and preferences. The system uses Machine Learning algorithms to provide accurate admission predictions and personalized college recommendations.

## 🚀 Features

* 🔐 User Registration and Login System
* 🤖 AI-Powered Admission Prediction
* 🏫 College Search and Comparison
* ⭐ Shortlist and Save Favorite Colleges
* 📊 Interactive Analytics Dashboard
* 💬 Chat and User Interaction Module
* 🖼️ College Information with Images
* 📈 Data Visualization using Plotly
* 🗄️ SQLite Database Integration
* 📱 User-Friendly Interface built with Streamlit

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* Plotly
* SQLite
* Pillow (PIL)

## 🎯 Project Objective

The main objective of this project is to simplify the college admission process by providing students with data-driven insights and predicting their admission possibilities using Machine Learning techniques.

## 💡 How It Works

1. User creates an account and logs in.
2. Student enters academic details and preferences.
3. The ML model analyzes the input data.
4. The system predicts admission chances and suggests suitable colleges.
5. Users can compare colleges, shortlist options, and track their predictions.
